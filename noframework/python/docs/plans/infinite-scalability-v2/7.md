## Acceptance Criteria

- [ ] Extracts `import X` statements
- [ ] Extracts `from X import Y, Z` statements
- [ ] Extracts relative imports (`from . import X`)
- [ ] Works for Python
- [ ] Works for JavaScript/TypeScript (import/require)
- [ ] Returns empty list for unsupported languages

## Language-specific Patterns

### Python
```python
import os                    # Import(module="os", names=[], line=1)
from os import path, getcwd  # Import(module="os", names=["path", "getcwd"], line=2)
from . import utils          # Import(module=".", names=["utils"], is_relative=True)
from ..core import main      # Import(module="..core", names=["main"], is_relative=True)
```

### JavaScript/TypeScript
```javascript
import axios from 'axios';           // Import(module="axios", names=["axios"])
import { get, post } from './http';  // Import(module="./http", names=["get", "post"])
const fs = require('fs');            // Import(module="fs", names=["fs"])
import * as utils from './utils';    // Import(module="./utils", names=["*"])
```
