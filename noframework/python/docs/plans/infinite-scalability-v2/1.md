# Task 1: Project Scaffolding

## Objective
Set up the basic project structure for the tech_writer module.

## Deliverables

```
tech_writer/
├── __init__.py
├── cli.py          # Entry point (stub)
├── store.py        # SQLite cache (stub)
├── tools/
│   ├── __init__.py
│   ├── filesystem.py   # list_files, read_file
│   └── semantic.py     # get_symbols, etc.
├── parser.py       # Tree-sitter integration (stub)
├── orchestrator.py # Exploration/generation loops (stub)
├── citations.py    # Citation parsing/verification (stub)
└── llm.py          # LLM interface (stub)
```

## Acceptance Criteria

- [ ] Directory structure exists
- [ ] All `__init__.py` files created with appropriate exports
- [ ] Each module has a docstring explaining its purpose
- [ ] `python -m tech_writer --help` runs without error (shows stub help)
- [ ] pyproject.toml updated with tech_writer entry point

## Implementation Notes

- Keep stubs minimal: just enough to import without error
- Use type hints from the start
- Follow existing project conventions (see tech-writer.py for style)
