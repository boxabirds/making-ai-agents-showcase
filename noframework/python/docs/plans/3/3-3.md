## Requirements

Update `tech_writer/cli.py`:

1. **Add `--skip-complexity` flag**
   ```python
   parser.add_argument(
       "--skip-complexity",
       action="store_true",
       help="Skip complexity analysis (use default budgets)",
   )
   ```

2. **Add `--dry-run` flag**
   ```python
   parser.add_argument(
       "--dry-run",
       action="store_true",
       help="Show complexity analysis without running pipeline",
   )
   ```

3. **Complexity analysis call**
   - After repo resolution, before pipeline
   - Skip if `--skip-complexity`
   - Log metrics to stderr

4. **Dry run behavior**
   - Display bucket, max_sections, max_exploration
   - Display top 5 complex functions
   - Exit without running pipeline
